# Tutta – Feature List

## MVP Features

These features are required for the first working version of Tutta.

### Authentication
- User registration
- User login
- User profile

### Listings
- View list of homes
- View listing details
- Add a new listing
- Upload listing images
- Set price (daily / weekly / monthly)

### Search and Filters
- Search listings
- Filter by price
- Filter by location
- Filter by room type

### Map
- View listings on map
- Show price pins on map

### Booking
- Select dates
- Check availability
- Create booking
- Cancel booking

### Chat
- Guest and host messaging
- Basic chat system

### Reviews
- Guest review host
- Host review guest

### Payments
- Click payment
- Payme payment

---
